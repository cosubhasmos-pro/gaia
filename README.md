# gaia
Analysis of Public Gaia DR3 datasets to understand galactic archaeology

# Environemt
* python3 -m venv gaiaenv
* python3 -m ipykernel install --user --name gaiaenv --display-name "Python (Gaia)"
You might need to install ipykernel to do so. Install it by "pip install ipykernel".

# Installation
* pip install -r requirements.txt
* python3 -m pip install --only-binary galpy galpy
